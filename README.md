# Projeto PI – TecSistem

## Status do projeto
Em desenvolvimento

## Tecnologias utilizadas
- Java
- Swing
- MySQL
- JDBC
- Git e GitHub

## Time de desenvolvedores
- Robson Souza

## Objetivo do software
Sistema desktop para gerenciamento de pedidos, atendimento, cozinha, caixa e financeiro.

## Funcionalidades
- Cadastro e listagem de pedidos
- Atualização de status dos pedidos
- Integração com banco de dados MySQL
- Controle de caixa e pagamentos
- Visualização financeira
